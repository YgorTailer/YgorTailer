## Agora

Estou criando a **Nivo**, uma plataforma de cloud hosting para hospedar apps, bots, databases e arquivos com uma experiência simples no painel e uma base forte de infraestrutura.

| Produto | Descrição |
| --- | --- |
| **Site** | Página pública e entrada principal em [nivo.lat](https://nivo.lat). |
| **Admin** | Painel operacional para nodes, regiões, projetos, deploys e usuários. |
| **API** | Backend central para autenticação, billing, deploy orchestration e integrações. |
| **Daemon** | Agent instalado nos nodes para provisionar containers, rede, segurança e métricas. |
| **CDN/Storage** | Entrega de arquivos e assets por [cdn.nivo.lat](https://cdn.nivo.lat). |

| **Produto** | Painel admin, CLI/SDK, fluxos de deploy e cloud management |

## Foco

- Construir uma cloud própria com fluxo profissional de criação de projetos, nodes e regiões.
- Automatizar infraestrutura sem deixar o usuário lidar com detalhes desnecessários.
- Criar uma experiência limpa para deploy, storage, logs, métricas e segurança.
